Welcome to another ExpressJS Boilerplate

This app has token authentication set up and ready to use. Each route, except for user authenticating, has token authorizing middleware set up to assess. By default, Tokens will expire every 3 days.

An example of handling token expiration: If a user accesses the client application within 6 hours of their token expiry, you can use the default refresh-token api route provided by this app to get a new token and retire the current one to a blacklisted token collection. The blacklist ensures that tokens discarded prior to expiry are no longer usable. Blacklisting also occurs on hitting the api's logout route.
